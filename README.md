# Instructions

Some steps to clone, check, and run the project:

# 1. Install dependencies (only needed once)
- follow the [instructions](https://gitlab.lnu.se/1dv528/course-content/html-css-javascript-dev-tools)
 to setup your envoirnment (if you haven't done it before)


# 2. Clone the repository
- Copy `git@gitlab.lnu.se:1dv528/student/sa225sd/a1-website.git` from this repo.
- Open terminal and navigate to the folder you want to put the repo in.
- Use `git clone` and paste the repo link.

# 3. Run all linters (HTML + CSS + JavaScript)
- To run linters use `npm run lint`


# 4. Start the web server
- To run the server use `npm run http-server`
- In the terminal after running `npm run http-server`, it should appear a local link to the website after a yellow/green text. 
- - Something like: `http://127.0.0.1:9001`or `http://192.168.8.113:9001`, click on it and it should take you to the website.
